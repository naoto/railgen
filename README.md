# Railgen

Railgen is generate gem that provides a rails application

## Installation

Add this line to your application's Gemfile:

    gem 'railgen'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install railgen

## Usage

### Generate Skeleton

```bash
$ railgen new appname [option]
```

### Option

- `-t`,`--target`: target directory
